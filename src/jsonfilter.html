JSONPath Guide

Basic Syntax

$ Root node
@ Current node
. Child operator
.. Recursive descent
* Wildcard
[] Subscript operator
[,] Union operator
[start:end] Array slice
[?()] Filter expression

Common Examples

$ Root object
$.store.book[*] All books
$.store.book[0] First book
$.store.book[-1] Last book
$.store.book[0:2] First two books
$.store.book[*].title All book titles
$.store.book[*].author All authors
$.store.book[?(@.price < 10)] Books under $10
$.store.book[?(@.price > 10)] Books over $10
$.store.book[?(@.category == 'fiction')] Fiction books only
$..author All authors (recursive)
$.store..price All prices in store
$.store.book[0,1] First and second books
$.store.* All direct children of store
